# Claude Code로 개발하는 방법

> CLI 기반 AI 코딩 도구 Claude Code를 실제 프로젝트에 적용한 경험을 정리합니다.

---

## Claude Code란?

Claude Code는 Anthropic이 만든 CLI 기반 AI 코딩 어시스턴트입니다.
터미널에서 직접 실행되며 코드베이스 전체를 컨텍스트로 이해하고 파일을 읽고 수정합니다.

단순한 코드 생성 도구가 아닌 **코드베이스를 탐색하고 수정하는 에이전트**입니다.

---

## 이 포트폴리오에서의 활용

이 사이트 자체가 Claude Code와 함께 개발되었습니다.

- 컴포넌트 구조 설계 및 구현
- Amplify 배포 오류 디버깅 (`gzipSync` 미import 등)
- Mermaid 렌더링 오류 원인 분석 및 수정
- 페이지네이션, 필터링 기능 추가

---

## 효과적인 활용 패턴

### 1. 오류 로그를 그대로 붙여넣기

```
Amplify 배포 중 발생한 오류야 문제를 해결해줘
[오류 로그 전체]
```

원인 분석부터 수정까지 한 번에 처리됩니다.

### 2. 기능 단위로 요청하기

```
Articles쪽에서 문제가 계속 발생하고 있어.
이미지는 안 나오도록 해주고, 페이지네이션을 추가해줘
```

세부 구현 방법을 지시하지 않고 **목적**을 말하면 적합한 방법을 선택합니다.

### 3. 계획 검토 후 실행

복잡한 기능은 먼저 계획을 요청하고 검토한 뒤 실행합니다.

```
AI를 사용하는 개발자라는 느낌이 들지 않는 것 같아.
이 포트폴리오 사이트 보완 계획을 작성해줘
```

계획을 검토하고 `데모페이지는 제외하고 수행해줘`처럼 조정할 수 있습니다.

---

## 한계와 주의점

- 생성된 코드가 동작해도 **이해한 것은 아닙니다**
- 핵심 로직(스트림 처리, 인증 흐름 등)은 직접 읽고 이해해야 합니다
- AI가 선택한 패턴을 수정하거나 디버깅하려면 해당 패턴을 알아야 합니다

> "요청을 잘 쓰는 것"과 "코드를 이해하는 것"은 별개입니다.

---

## 교훈

Claude Code는 개발 속도를 높이는 강력한 도구지만,
코드 오너십은 개발자가 유지해야 합니다.

AI가 만든 코드를 검토하고, 이해하고, 필요하면 수정하는 능력이 핵심입니다.
